# Trip with Me Travel — Website

Landing page resmi Trip with Me Travel (PT. Sinar Abadi Arifin), Makassar.

- **Stack:** HTML + CSS + JS statis (tanpa build tool)
- **Font:** Montserrat Alternates + Outfit (Google Fonts)
- **Warna brand:** #287094 / #0867a8

## Struktur
- `index.html` — seluruh halaman
- `assets/logo/` — logo
- `assets/img/` — foto perjalanan

## Deploy ke GitHub Pages
1. Buat repo baru di GitHub (mis. `tripwithme-web`).
2. Upload semua isi folder ini ke repo.
3. Settings → Pages → Source: `main` / root → Save.
4. Situs live di `https://<username>.github.io/tripwithme-web/`.
